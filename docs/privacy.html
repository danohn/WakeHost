Privacy Policy

WakeHost Privacy Policy

Last updated: March 12, 2026

WakeHost is designed to work locally on your Mac. It does not include analytics, advertising, or third-party tracking.

Information WakeHost uses

WakeHost stores the following information on your device so the app can connect to your OPNsense server:

  • OPNsense server address
  • OPNsense server port
  • API key
  • API secret

How information is stored

The server address and port are stored in local app preferences on your Mac. The API key and API secret are stored locally in Keychain.

How information is used

WakeHost uses this information only to make requests to the OPNsense API that you configure. These requests are used to fetch Wake-on-LAN hosts and send wake requests.

Data sharing

WakeHost does not sell, share, or transmit your information to any third-party service other than the OPNsense server you configure.

Analytics and tracking

WakeHost does not use analytics, advertising SDKs, or cross-app tracking.

Local network access

WakeHost requests local network access on macOS so it can reach your OPNsense router when it is available on your local network.

Contact

If you have questions about this privacy policy, use the project repository at github.com/danohn/WakeHost.
