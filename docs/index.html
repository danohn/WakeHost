macOS Menu Bar Utility

Wake devices through OPNsense from your menu bar.

WakeHost is a lightweight macOS app for refreshing Wake-on-LAN hosts and sending wake requests without keeping a full window open.

View on GitHub Privacy Policy

Menu bar first

Refresh hosts, wake devices, and open settings directly from a compact popover.

Guided setup

First-run onboarding helps users enter their OPNsense address, port, and API credentials.

Local credentials

API credentials are stored on-device in Keychain and used only for OPNsense API requests.

Requirements

WakeHost requires a macOS device and an OPNsense installation with Wake-on-LAN hosts configured.
